## 使用
1. 解壓縮後，整包上傳到 GitHub Pages / Netlify。
2. 用手機打開 `index.html` 所在網址 → 可加到主畫面。
3. 下方輸入框貼上文章（可含「分類：」「標題：」）→ 按「送出」，資料會寫入 Google 試算表（Notebook 分頁）。
