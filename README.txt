
Notebook PWA v1.0.3（中文動作固定版）

已預設：
- API：https://script.google.com/macros/s/AKfycbx3oTDF49EmwGp5ZgGs9WIO64m7UzVzj5IDcBmm222aR7eTQTG8DXUsYVGVnzH2ga4abQ/exec
- 動作：列出筆記 / 取得筆記 / 新增筆記 / 修改筆記 / 刪除筆記
- 欄位鍵名：標題 / 內容 / 分類 / 標籤 / 日期 / id
- 搜尋會同時送 keyword 與 q
- 右滑返回上一頁；禁用雙擊與捏合放大
- 頂端會顯示載入筆記的筆數與最近同步時間，背景輪詢時會標示「背景更新中…」

若你重新部署 GAS，請打開 index.html 搜尋 const API，把網址換成新部署。

操作方式：
1. 點「更新」可手動同步，若顯示「背景更新…」代表系統正在自動輪詢。
2. 使用搜尋列輸入關鍵字再按「搜尋」，清除後按「清空」。
3. 點選筆記或右下角「＋ 新增」進入編輯視窗，可儲存或刪除。
4. 在內容中輸入「標題：」「分類：」「標籤：」會自動帶入欄位，沒有標題會改取前八個字。
5. 編輯頁面下方的狀態列會顯示儲存結果；完成後清單會自動重新整理。

測試：
- 安裝依賴：`npm install`
- 執行單元測試：`npm test`
